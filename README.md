# Portfolio Site - DESENDEV

Site portfolio pessoal desenvolvido com HTML, CSS e JavaScript puro.

## Estrutura do Projeto

- `index.html` - Página principal
- `about.html` - Página sobre
- `work.html` - Página de detalhes dos projetos
- `styles.css` - Estilos do site
- `script.js` - Funcionalidades JavaScript
- `images/` - Imagens e assets do projeto

## Tecnologias Utilizadas

- HTML5
- CSS3
- JavaScript (Vanilla)
- Fontes do Google Fonts (Inter)

## Projetos em Destaque

- **SetYou** - Loja virtual completa hospedada no Shopify
- **Vumbora** - Plataforma de passeios turísticos

## Deploy

O site está hospedado no Netlify.
